Labeller

Labeller is a minimalistic labelling tool for files containing time series data.

Quick start

  1. Load a dataset. See Dataset format for information on the dataset format.
  2. Set a dataset visualisation format by specifying a format string.
  3. Set the null class values.
  4. Select Apply.
  5. Select which is the column containing the label channel and press Confirm label.
  6. Annotate the data.
  7. Save a project or export the annotated data.

The software offers additional features:

  • A new data channel can be added from the Edit menu. This may be used to add an annotation channel to a data file.

Dataset format

Datasets are text files containing a matrix of data separated by space and newline. Each line corresponds to a sample. Each column corresponds to a data channel.

The labels of the data are stored as one channel of the dataset. Each sample can potentially have a different label. In practice, however, labelling highlights instances which are contiguous segments of data tagged with the same label.

Visualization format

The way the signal traces are displayed are specified by a display format string. This string indicates how many scopes are on screen and which traces they should show.

The syntax is: <title;traces[;xscale=xs][;yscale=auto|min max][;color=colors]><......

  • title: title of the scope
  • traces: space separated list of traces to display (0=first trace, ...)
  • xscale: horizontal scale. 1=default. Positive value: zoom in. Negative value: zoomout
  • yscale: vertical scale. automatic (auto) or manual (min max)
  • colors: space separated list of trace colors in hexadecimal format RRGGBB

Example

The format string below shows two scopes. Each scope contains 3 traces: the first contains the traces 0, 1, and 2, and the second contains the traces 3, 4, and 5.

<Acceleration;0 1 2;yscale=-4000 4000;color=ff0000 00ff00 0000ff> <Gyroscope;3 4 5;yscale=auto>

The following display format string displays two scopes. The first shows traces number 5, 6 and 7, in red, green and blue respectively, with manual vertical scale, default horizontal scale, and the title Calibrated.

<Calibrated;5 6 7;xscale=1;yscale=-3300 3300 ;color=ff0000 ff00 ff><Raw;2 3 4;xscale=-2;yscale=auto ;color=ff0000 ff00 ff>

Null class

The data instances (i.e. contiguous samples with the same label value) are rendered differently to distinguish null and non-null instances.

Null instances are not outlined; as if they were a non-interesting part of the dataset. Non-null instances, however, are outlined. The timespan of non-null instances can be edited through this software.

The labels corresponding to null can be specified in the Null class labels field. Multiple null labels can be specified. The first null label is the one used by default when an instance is resized or erased.

Data annotation

The data can be annotated as follows:

  • The edges of an instance can be grabbed and moved by a mouse right-click and drag.
  • A new instance can be added by a mouse right-click and drag. This is possibly anywhere far from an existing instance edge.
  • An existing instance can be deleted by pressing the Del key while hovering over the instance
